# DSA – Learning by Doing (JavaScript)

This repository documents my hands-on journey learning Data Structures and Algorithms using JavaScript.

The focus is on **implementation from scratch**, **problem-solving**, and **understanding trade-offs** — not memorizing solutions or farming LeetCode points. Each data structure and algorithm is written manually, tested, and iterated on as I learn.

This is a work-in-progress. Code may evolve, get refactored, or be rewritten as my understanding improves.

## Rules

- No copy-pasting solutions without understanding and rewriting
- Every implementation must be written from scratch
- Revisit and refactor old code as understanding improves
- Prioritize clarity over cleverness
